# Module cryptography-serialization-asn1-modules

Pre-defined ASN.1 modules/declarations from cryptography-related RFCs.

The module provides common ASN.1 structures including:

- [SubjectPublicKeyInfo][dev.whyoleg.cryptography.serialization.asn1.modules.SubjectPublicKeyInfo]
  from [RFC 5280](https://datatracker.ietf.org/doc/html/rfc5280)
- [PrivateKeyInfo][dev.whyoleg.cryptography.serialization.asn1.modules.PrivateKeyInfo]
  from [RFC 5208](https://datatracker.ietf.org/doc/html/rfc5208)
