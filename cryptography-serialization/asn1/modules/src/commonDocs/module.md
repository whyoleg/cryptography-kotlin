# Module cryptography-serialization-asn1-modules

Pre-defined ASN.1 modules/declarations from cryptography-related RFCs.

The module provides common ASN.1 structures including:

- [SubjectPublicKeyInfo][SubjectPublicKeyInfo] from [RFC 5280][RFC5280]
- [PrivateKeyInfo][PrivateKeyInfo] from [RFC 5208][RFC5208]
