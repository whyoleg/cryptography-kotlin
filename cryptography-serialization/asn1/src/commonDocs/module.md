# Module cryptography-serialization-asn1

Binary serialization of ASN.1/DER content via [kotlinx.serialization][kotlinx.serialization].

The module provides [DER][DER] format for encoding and decoding ASN.1 structures.

> ASN.1 functionality is currently experimental and is subject to API/ABI changes
